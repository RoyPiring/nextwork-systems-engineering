### What is DevOps?

DevOps is an operating model that treats software delivery and system reliability as a single responsibility. It relies on automation for provisioning, deployment, and testing, and embeds monitoring, incident response, and security controls directly into the delivery lifecycle.

### What is CI/CD?

CI/CD is an automated delivery system that controls how code moves from commit to deployment. It standardizes integration, testing, and release steps to reduce human error, shorten feedback loops, detect defects early, and coordinate work across development, operations, and security teams.
