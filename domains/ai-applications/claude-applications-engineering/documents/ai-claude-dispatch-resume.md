## Introducing Today's Project!

In this project, I built a phone-to-desktop workflow that tailors a resume to a specific job using Claude Dispatch. The goal was to reduce manual effort in resume customization while improving alignment with job requirements.

The workflow connects mobile input with desktop execution, allowing job descriptions to be processed in real time and translated into targeted resume updates.

### Key tools and concepts

This project used Claude Desktop, the Claude mobile app, and Claude Dispatch to enable cross-device interaction.

The concepts applied include AI-assisted resume optimization, device-to-device workflows, and reusable skill design for structured output generation.

### Challenges and wins

This project took about 45 minutes. The main challenge was establishing a stable connection between devices and configuring permissions correctly.

Once connected, the workflow executed reliably. The key win was creating a repeatable system that reduces friction in tailoring resumes for different roles.

### Why I did this project

The goal was to automate resume customization and reduce the time required to align experience with job descriptions.

This approach improves consistency and ensures that each resume version reflects the requirements of the target role.
