## Introducing Today's Project!

In this project, I explored three Claude tools, Claude.ai, Claude Code, and Claude Cowork, by analyzing a cloud salary dataset. The goal was to understand how each tool supports different parts of an AI-assisted workflow including data exploration, visualization, coding, and project organization.

The exercise focused on using Claude.ai for conversational data analysis, Claude Code for building a dashboard through code, and Claude Cowork for organizing project outputs. This provided a quick view of how these tools can support data-driven analysis within a technical workflow.

### Key tools and concepts

This project used Claude.ai for conversational data exploration and quick visualization of uploaded datasets. Claude Code was used to interact with the data through a coding interface and generate an interactive dashboard. Claude Cowork was used to organize project artifacts and generate a summary of the analysis.

The concepts demonstrated in this exercise included data exploration, visualization, AI-assisted coding, and lightweight project organization. The focus was understanding how different AI tools support different stages of a data workflow.

### Challenges and wins

This project took about 45 minutes to complete. The main challenge was understanding how each Claude tool fits into the workflow and transitioning between the conversational interface and the coding environment.

Once the role of each tool became clear, the workflow was straightforward. Claude.ai handled exploration and chart generation, Claude Code handled dashboard creation, and Cowork handled project organization.

---

## Setting Up with Claude.ai and the Dataset

The first step was signing in to Claude.ai and preparing the dataset used for analysis. The cloud salary dataset served as the source of information for the entire exercise.

Uploading the dataset allowed Claude.ai to interpret the data structure and begin analyzing it conversationally. This enabled quick exploration of the dataset without requiring manual preprocessing.

![Image](http://learn.nextwork.org/refreshed_maroon_timid_jujube/uploads/ai-claude-compare_p3w8n1y6)

### Understanding the data

The dataset included columns representing job title, location, years of experience, salary, company size, and benefits package.

This structure allows analysis of compensation trends across roles, experience levels, and geographic regions. The data can also reveal how company size and experience influence salary ranges within cloud technology roles.

---

## Exploring Cloud Salaries with Claude.ai

Claude.ai was used to explore the dataset through conversational prompts. Uploading the dataset allowed questions to be asked directly about salary trends and role comparisons.

Artifacts were generated to visualize patterns in the data. These visualizations made it easier to identify trends such as salary differences across roles and experience levels.

![Image](http://learn.nextwork.org/refreshed_maroon_timid_jujube/uploads/ai-claude-compare_p3w8n5y1)

### Insights from the data

The analysis showed that roles such as Cloud Architect and AI Engineer consistently appeared among the highest paying positions within the dataset.

These roles typically command higher salaries due to the advanced technical skills required and the demand for architecture and AI expertise in cloud environments.

## Generating Interactive Charts with Artifacts

Artifacts in Claude.ai generated charts based on the dataset. These visualizations transformed the raw data into charts that made trends easier to interpret.

This approach allowed quick exploration of salary patterns without writing code, making it useful for rapid analysis and presentation.

![Image](http://learn.nextwork.org/refreshed_maroon_timid_jujube/uploads/ai-claude-compare_d9t1h7b3)

## Building a Dashboard with Claude Code

The next step involved installing Claude Code and accessing the dataset through the terminal environment.

Using Claude Code allowed the dataset to be analyzed programmatically. An interactive dashboard was generated using Chart.js, enabling dynamic visualization of the salary data.

![Image](http://learn.nextwork.org/refreshed_maroon_timid_jujube/uploads/ai-claude-compare_r8t2w4y6)

### How Claude Code works differently

Claude Code provides a coding interface that allows direct interaction with data through scripts and libraries. Unlike the conversational interface of Claude.ai, this approach gives more control over analysis and visualization.

Working through code makes it possible to build custom dashboards, automate analysis tasks, and modify how data is processed and displayed.

![Image](http://learn.nextwork.org/refreshed_maroon_timid_jujube/uploads/ai-claude-compare_d7f9h1j3)

### The interactive dashboard

Claude Code generated an interactive dashboard that visualized salary trends using Chart.js. The dashboard allowed exploration of the dataset through dynamic charts.

This approach provided more customization than the conversational interface and demonstrated how AI-assisted coding can quickly produce data visualization tools.

---

## Organizing Everything with Claude Cowork

Claude Cowork was used to organize the files created during the project. The project folder was shared with Cowork so it could index the files and generate a summary of the work completed.

Cowork organized the dataset, generated charts, dashboard output, and summary documentation into a structured project workspace.

![Image](http://learn.nextwork.org/refreshed_maroon_timid_jujube/uploads/ai-claude-compare_v4d3e4f5)

### What makes Cowork unique

Cowork focuses on project organization rather than data analysis. While Claude.ai supports conversational exploration and Claude Code supports coding workflows, Cowork centralizes project artifacts and generates summaries of work performed.

This makes it useful for managing files and documenting project outcomes.

![Image](http://learn.nextwork.org/refreshed_maroon_timid_jujube/uploads/ai-claude-compare_v4j9k0l1)

### The organized project folder

Cowork generated a summary document describing the dataset analysis, the charts created, and the dashboard built with Claude Code.

This document provided a concise overview of the project and served as a reference for the steps performed and insights discovered during the exercise.

## Building a Claude Tool Cheat Sheet

Using Cowork, I created a simple reference describing when to use each Claude tool during a workflow.

Claude.ai is useful for quickly exploring datasets and generating visualizations through conversational prompts. Claude Code is useful when deeper control is required for analysis, scripting, or building dashboards. Cowork is useful for organizing files and generating project summaries once the work is complete.

![Image](http://learn.nextwork.org/refreshed_maroon_timid_jujube/uploads/ai-claude-compare_p4n6x1j9)

### My decision framework

I used Claude.ai, Claude Code, and Cowork together because each tool serves a different role in the workflow.

Claude.ai supports fast exploratory analysis. Claude Code enables programmatic data analysis and dashboard creation. Cowork manages project organization and documentation. Using all three tools together creates a simple workflow for exploring data, building visualizations, and organizing results.
